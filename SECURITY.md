# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do NOT open a public issue.**

Instead, use [GitHub Security Advisories](https://github.com/jadecli/claude-knowledge-sdk-typescript/security/advisories/new) to report the vulnerability privately.

We will acknowledge within 48 hours and provide a timeline for fixes.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
